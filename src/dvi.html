3D Viewer

About programm

The program is implemented on the gkt3+ graphics library and C11 standart. The viewer allow you to show .obj file, rotate, move, scale them.

Also emplements ability to save screenshot and make gif (5 sec 10 fps).

Install

Move to src folder and run commands

make install

Run

Move to src folder and run command

make viewer_app

Unistall

Move to src folder and run command make uninstall

make uninstall
